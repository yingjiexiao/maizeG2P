## readme.txt

1. System requirements

The "normalization.r" is a user-friendly R code which can be adapted to any R version without any packages dependencies

2. Intallation guide

Our code don't need to install any dependency R packages

3 and 4. Demo and Intructions for use

We provide one R scripts: normalization.r. 
All things need to do is run the "normalization.r", in which, the program will import three sample data "MJ1221.txt", "MZ1221.txt", "pop6210.txt" and export the normalized phenotypes in "pop8652.Nor.txt" file.
